Docker学习笔记

一、windows10安装Docker

开启Hyper-V

  1. windows10直接搜索【启用或关闭windows功能】
  2. 勾选Hyper-v，然后点击确定即可

下载Docker安装包

  1. 安装包下载地址为https://hub.docker.com/editions/community/docker-ce-desktop-windows
  2. 点击安装包，然后按照提示直接安装即可

检查是否安装成功

Centos7安装Docker需要64位系统，内核版本3.10以上

四、Docker常用命令

列出本机所有image文件

Docker 把应用程序及其依赖，打包在 image 文件里面。image 文件可以看作是容器的模板，Docker根据 image 文件生成容器的实例，同一个 image 文件，可以生成多个同时运行的容器实例

查看本机所有image文件：

删除image文件

从docker仓库抓取image文件

运行image文件

运行image文件时，会生成一个正在运行的容器实例。

OPTIONS:

-d: 后台运行容器，并且返回容器ID

-i: 以交互模式运行容器，通常与 -t 同时使用

-P: 随机端口映射，容器内部端口随机映射到主机端口

-p: 指定端口映射，后面接的数据为主机端口：容器端口

-t: 为容器重新分配一个伪输入终端，通常与 -i 同时使用

–name: 为容器指定一个名称

-e: 设置环境变量

-v: 挂载主机目录到容器，格式为-v src_path:dest_path

iamgeName名称后面可以紧接着需要在容器中执行的命令。
